### Part 2: What Makes Claude Code Different (30 minutes)
**Goal:** Showcase unique, advanced features

**Demo 5: Skills - Persistent Expertise (10 minutes)**

- Demonstrate custom skill activation (Spring Boot standards)

- Explain enterprise use case (50 developers, consistent standards)

**Demo 7: Hooks - Quality Gates (7 minutes)**
- Live demo: Security validator hook blocking hardcoded secrets
- Show auto-formatting hook improving code quality
- Explain SessionEnd hooks for summaries
- **Value prop:** Automated validation and workflow safety

**Demo 8: Subagents & Plan Mode (5 minutes)**
- Activate Plan Mode (Shift+Tab+Tab)
- Show Plan subagent creating strategy
- Explain other subagents (Explore, Testing, Documentation)
- **Value prop:** Intelligent task specialization

**Key Takeaway:** These features enable enterprise-grade AI-assisted development

## Presenter Notes

### Energy & Pacing
- Keep demos moving - 90 minutes goes fast
- Have backup examples ready in case of technical issues
- Build excitement through progressive reveals
- "Wow moments" when showing advanced features

### Engagement Strategies
- Poll questions during transitions
- "Raise hand if you've experienced [common pain point]"
- Encourage questions in chat throughout
- Share personal stories/experiences with the tool

### Technical Contingencies
- Pre-record critical demos as backup
- Have screenshots ready if live demo fails
- Test all demo projects 1 hour before webinar
- Keep terminal history clean for easy re-running

### Time Management
- Set timer for each section
- If running long, cut "Advanced Features Overview" first
- Protect Q&A time - it's where real value often emerges
- Have "bonus demos" ready if running short
